PanoroCRM

Period involved:
2017 - 2022
Tech stack:
NodeJS, ReactJS, C++
My involvement:

PanoroCRM is a complex modular CRM with multiple modules for managing real estate offers, publishing them to different real estate portals, create custom shareable websites with offers to be sent to the clients, adding clients and keeping track of them and their interests and seeing statistics about the activity done.

  • complex application composed of many modules and services;
  • front-end created with ReactJS;
  • back-end created using Node.js;
  • microservices architecture;
  • critical services created with C++ for best performance.

Some modules integrated:

  • real estate ads module that is responsible for the adding/editing and updating of users' ads;
  • contacts module offers the users the ability to add contacts, link them to ads for better organization;
  • history module provides users logs of their whole activity on the CRM;
  • agencies module provides the agencies administrators ability to add agents, limit the number of ads they can add and track their activity;
  • integration with imobiliare.ro, storia.ro/olx.ro and panoro.ro - giving users the ability to add their ads directly from the CRM to those platforms.
