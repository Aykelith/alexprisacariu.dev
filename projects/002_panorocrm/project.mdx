PanoroCRM is a complex modular CRM with multiple modules for managing real
estate offers, publish them to different real estate portals, create custom
shareable websites with offers to be sent to the clients, add clients and keep
track of them and their interests and see statistics about the activity done.

- complex application composed of many modules and services;
- front-end created with ReactJS;
- back-end created using Node.js;
- microservices architecture;
- critical services created with C++ for best performance.

Some modules integrated:

- real estate ads module that is responsible for the adding/editing and
  updating of users' ads;
- contacts module offers the users the ability to add contacts, link them
  to ads for better organization;
- history module provides users logs of their whole activity on the CRM;
- agencies module provides the agencies administrators ability to add agents,
  limit their number of ads that can add and track their activty;
- integration with 
  <a href="https://www.imobiliare.ro/" target="_blank" rel="noreferrer noopener">imobiliare.ro</a>,
  <a href="https://www.storia.ro/" target="_blank" rel="noreferrer noopener">storia.ro</a>/<a href="https://www.olx.ro/" target="_blank" rel="noreferrer noopener">olx.ro</a>
  and <a href="https://alexprisacariu.dev/project/003_panoro.ro" target="_blank" rel="noreferrer noopener">panoro.ro</a> - 
  adding the ability to the users to add their ads directly from the CRM to
  those platforms.
